##4、网址名，网页名
    1、关于 include()的使用

    3、render如何找到网页？

    4、处理方法：在每个app下面创建一个templates/app的文件夹，用来存放，

    5、处理突然想要修改网址：
        5.1、给网址起名字 path('xxx/', views.xxx, name='xxx')
        <a href="{% url 'todolist_about' %}"></a>
        建议：name = 'appname_xxx'
    6、网址名和其他app里面的重复了，处理办法
        给一个前缀:
        urls.py -> app_name = 'appname'
        <a href="{% url 'appname:todolist_about' %}"></a>
